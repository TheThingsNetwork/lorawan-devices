About this device

IWM-LR3 has been designed to allow wireless remote reading in different types of applications from the residential sector to the commercial and industrial sectors. It can read the inductive target of the meter dial using a pulse sensor.

The IWM-LR3 - Water Meter Reader supports 1 regional frequency plan and operates as a Class A LoRaWAN device.

Key features

  • Sensors: pulse count.
  • LoRa Alliance certified, verified interoperability with The Things Stack and other LoRaWAN network servers.
  • 1 frequency plan supported: EU863-870.
  • Battery powered (3V Lithium).
