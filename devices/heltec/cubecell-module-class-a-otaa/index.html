About this device

The HelTec HTCC-AM01(Class A OTAA) CubeCell – Module is a LoRaWAN® module based on ASR605x (ASR6501, ASR6502) and integrated with the PSoC® 4000 series MCU (ARM® Cortex® M0+ Core) and SX1262. This module supports the Arduino® development environment.

The HTCC-AM01(Class A OTAA) supports 8 regional frequency plans and operates as a Class A LoRaWAN device.

Key features

  • 8 frequency plans supported: AS923, AU915-928, CN470-510, EU433, EU863-870, IN865-867, KR920-923, US902-928.
