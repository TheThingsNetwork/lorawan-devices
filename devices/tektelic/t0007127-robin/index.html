About this device

TEKTELIC’s ROBIN is a compact, mobile, wireless outdoor BLE panic button that transmits an emergency signal from any location. When triggered, this device instantly sends an SOS message containing location details to designated support staff or security personnel to ensure a rapid response in the event of a panic situation.

The ROBIN Outdoor Panic Button supports 7 regional frequency plans and operates as a Class A LoRaWAN device. A payload codec is published in the repository, so uplinks are decoded automatically when you onboard it to The Things Stack.

Key features

  • Sensors: button.
  • Payload codec included, uplink decoder ready for one-click activation.
  • 7 frequency plans supported: AS923, AU915-928, EU863-870, IN865-867, KR920-923, RU864-870, US902-928.
  • Battery powered (C Cell).

Manual setup

1
Create an application

In The Things Stack console, go to Applications → Create application and pick a unique application ID.

2
Register an end device

Choose From the LoRaWAN Device Repository and search for “ROBIN Outdoor Panic Button”. Select your firmware version and frequency plan.

3
Enter device credentials

Provide the DevEUI, JoinEUI and AppKey printed on the device label or QR code.

4
Power on & verify uplinks

Within a few minutes you'll see the join request and decoded uplinks in the Live data tab.
