About this device

The Noise Level Monitoring Station is a hardware noise monitoring solution that helps cities and communities sense accurate noise level data. The Station operates autonomously without requiring external power source or manual data gathering.

The Urban Noise Monitoring Station supports 9 regional frequency plans and operates as a Class A LoRaWAN device.

Key features

  • Sensors: sound, temperature, battery.
  • LoRa Alliance certified, verified interoperability with The Things Stack and other LoRaWAN network servers.
  • 9 frequency plans supported: AS923, AU915-928, CN470-510, CN779-787, EU863-870, IN865-867, KR920-923, RU864-870, US902-928.
  • Battery powered (Lithium), user replaceable.
