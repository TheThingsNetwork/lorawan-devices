R718PE - Wireless Top Mounted Ultrasonic Liquid Level Sensor

By Netvox Technology Co., Ltd

R718PE is a wireless communication device that uses ultrasound to detect the distance between the device and the detected object. The medium the device detects is the air, so the detected object can be any liquid or solid that has a flat and horizontal surface, water level for example. The main body and the sensor of R718PE communicates via the UART serial port, and the detected data can be transmitted via LoRa communication.
