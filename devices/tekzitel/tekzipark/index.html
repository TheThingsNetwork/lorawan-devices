About this device

The TEKZITEL TEKZIPARK is a smart parking sensor that uses radar technology. It enables occupancy status updates of parking spots wirelessly to the cloud environment through a LoRaWAN® network. This helps to efficiently manage the parking spaces in real-time.

The TEKZIPARK supports 3 regional frequency plans and operates as a Class A LoRaWAN device. A payload codec is published in the repository, so uplinks are decoded automatically when you onboard it to The Things Stack.

Key features

  • Sensors: radar, magnetometer, temperature, battery.
  • Payload codec included, uplink decoder ready for one-click activation.
  • 3 frequency plans supported: AU915-928, EU863-870, US902-928.
  • Battery powered (C), user replaceable.
