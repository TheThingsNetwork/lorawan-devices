Animal Activity Sensor

By MCCI Corporation

The MCCI Catena® 4430 Activity Sensor Wing allows mounting Panasonic WL series PIR sensors and connecting additional external sensors. It can be used with MCCI Catena 4610, Catena 4612/4618, and the Adafruit Feather M0 with LoRa® radio system, but suitable for use with any Adafruit Feather CPU board.
