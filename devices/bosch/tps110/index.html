About this device

The BOSCH wireless smart parking sensor is a LoRaWAN® end device that uses a magnetometer and radar which are two independent sensors for parking space occupancy. It enables active parking lot management features, such as search, navigation, and reservation. In addition to that, the BOSCH parking lot sensor has a GPS receiver and a temperature sensor.

The Parking Lot Sensor supports 2 regional frequency plans and operates as a Class A LoRaWAN device. A payload codec is published in the repository, so uplinks are decoded automatically when you onboard it to The Things Stack.

Key features

  • Sensors: magnetometer, gps, temperature, radar.
  • LoRa Alliance certified, verified interoperability with The Things Stack and other LoRaWAN network servers.
  • Payload codec included, uplink decoder ready for one-click activation.
  • 2 frequency plans supported: AS923, EU863-870.
  • Battery powered (Lithium battery (Li-SOCI2)).
