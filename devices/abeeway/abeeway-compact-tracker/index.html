About this device

The Abeeway Compact Tracker is a highly versatile multi-mode tracker for outdoor and indoor geolocation with embedded sensors combining GPS, Low-power GPS, Wi-Fi Sniffer, BLE, and LoRa TDoA geolocation technologies.

The Compact Tracker supports 3 regional frequency plans and operates as a Class A LoRaWAN device. A payload codec is published in the repository, so uplinks are decoded automatically when you onboard it to The Things Stack.

Key features

  • Sensors: accelerometer, gps, temperature.
  • LoRa Alliance certified, verified interoperability with The Things Stack and other LoRaWAN network servers.
  • Payload codec included, uplink decoder ready for one-click activation.
  • 3 frequency plans supported: AS923, EU863-870, US902-928.
  • Battery powered (3x AA/3.6V Li-SOCl2), user replaceable.

Manual setup

Create an application

In The Things Stack console, go to Applications → Create application and pick a unique application ID.

2
Register an end device

Choose From the LoRaWAN Device Repository and search for “Compact Tracker”. Select your firmware version and frequency plan.

3
Enter device credentials

Provide the DevEUI, JoinEUI and AppKey printed on the device label or QR code.

4
Power on & verify uplinks

Within a few minutes you'll see the join request and decoded uplinks in the Live data tab.
