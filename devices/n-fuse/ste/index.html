About this device

Environment sensor. BME680 (IAQ, CO2-e, bVOC-e, pressure, temperature, humidity).

The Environment sensor supports 12 regional frequency plans and operates as a Class A LoRaWAN device. A payload codec is published in the repository, so uplinks are decoded automatically when you onboard it to The Things Stack.

Key features

  • Sensors: battery, temperature, humidity, iaq, co2, bvoc, pressure.
  • Payload codec included, uplink decoder ready for one-click activation.
  • 12 frequency plans supported: AS923, AS923-2, AS923-3, AS923-4, AU915-928, CN470-510, EU433, EU863-870, IN865-867, KR920-923, RU864-870, US902-928.
  • Battery powered (CR2430), user replaceable.

Manual setup

1
Create an application

In The Things Stack console, go to Applications → Create application and pick a unique application ID.

2
Register an end device

Choose From the LoRaWAN Device Repository and search for “Environment sensor”. Select your firmware version and frequency plan.

3
Enter device credentials

Provide the DevEUI, JoinEUI and AppKey printed on the device label or QR code.

4
Power on & verify uplinks

Within a few minutes you'll see the join request and decoded uplinks in the Live data tab.
