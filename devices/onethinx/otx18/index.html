About this device

Onethinx Core LoRaWAN® module integrates Cypress® PSoC® 6 MCU, Semtech® SX126x chipset, dual ARM® Core processor (150MHz / 1MB Flash / 256KB SRAM), compact and highly efficient integrated antenna, and Bluetooth BLE 5.0. It provides faster development of LoRaWAN® products and also reduces development costs.

The OTX-18 LoRaWAN module supports 2 regional frequency plans and operates as a Class A LoRaWAN device.

Key features

  • LoRa Alliance certified, verified interoperability with The Things Stack and other LoRaWAN network servers.
  • 2 frequency plans supported: EU863-870, US902-928.
