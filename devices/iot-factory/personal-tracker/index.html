About this device

LORAWAN Personal Geolocation and Activity tracker

The Personal Tracker supports 4 regional frequency plans and operates as a Class A LoRaWAN device. A payload codec is published in the repository, so uplinks are decoded automatically when you onboard it to The Things Stack.

Key features

  • Sensors: gps, wifi ssid, accelerometer, button.
  • Payload codec included, uplink decoder ready for one-click activation.
  • 4 frequency plans supported: AS923, EU863-870, RU864-870, US902-928.
  • Battery powered (Lithium Polymer).

Manual setup

1
Create an application

In The Things Stack console, go to Applications → Create application and pick a unique application ID.

2
Register an end device

Choose From the LoRaWAN Device Repository and search for “Personal Tracker”. Select your firmware version and frequency plan.

3
Enter device credentials

Provide the DevEUI, JoinEUI and AppKey printed on the device label or QR code.

4
Power on & verify uplinks

Within a few minutes you'll see the join request and decoded uplinks in the Live data tab.
