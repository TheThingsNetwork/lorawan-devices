MultiTech xDot® Long Range LoRa Module

About this device

The MultiTech xDot® is a secure, CE/FCC/RCM/GITEKI certified, Arm® Mbed™ programmable, low-power RF module, that provides long-range, low bit rate M2M data connectivity to sensors, industrial equipment, and remote appliances. The xDot is LoRaWAN® 1.0.2 compliant, providing bi-directional data communication up to 10 miles/15 km line-of-sight and 1-3 miles / 2 km into buildings, using sub-GHz ISM bands in North America, Europe, Australia (AU915), Asia Pacific (AS923), India (IN865) and Korea (KR920).

The MultiTech xDot® Long Range LoRa Module supports 6 regional frequency plans and operates as a Class A/C LoRaWAN device.

Key features

  • LoRa Alliance certified, verified interoperability with The Things Stack and other LoRaWAN network servers.
  • 6 frequency plans supported: AS923, AU915-928, EU863-870, IN865-867, KR920-923, US902-928.
