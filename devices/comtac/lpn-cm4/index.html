About this device

The comtac LPN CM-4 SENSOR has a built-in temperature sensor and a humidity sensor that records temperature and moisture at programmable intervals. The collected data is sent over the LoRaWAN® network for further processing.

The LPN CM-4 Sensor - Temperature & Humidity Sensor supports 1 regional frequency plan and operates as a Class A LoRaWAN device. A payload codec is published in the repository, so uplinks are decoded automatically when you onboard it to The Things Stack.

Key features

  • Sensors: humidity, temperature, battery.
  • LoRa Alliance certified, verified interoperability with The Things Stack and other LoRaWAN network servers.
  • Payload codec included, uplink decoder ready for one-click activation.
  • 1 frequency plan supported: EU863-870.
  • Battery powered (LiMn02 1.5Ah AA battery), user replaceable.

Manual setup

2
Register an end device

Choose From the LoRaWAN Device Repository and search for “LPN CM-4 Sensor - Temperature & Humidity Sensor”. Select your firmware version and frequency plan.

3
Enter device credentials

Provide the DevEUI, JoinEUI and AppKey printed on the device label or QR code.

4
Power on & verify uplinks

Within a few minutes you'll see the join request and decoded uplinks in the Live data tab.
