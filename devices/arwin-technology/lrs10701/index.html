About this device

Indoor Air Quality (IAQ) Sensor

The LRS10701 - Indoor Air Quality (IAQ) Sensor supports 3 regional frequency plans and operates as a Class A LoRaWAN device. A payload codec is published in the repository, so uplinks are decoded automatically when you onboard it to The Things Stack.

Key features

  • Sensors: iaq, temperature, humidity, co2, tvoc, particulate matter, h2s, co, no2, tvoc.
  • Payload codec included, uplink decoder ready for one-click activation.
  • 3 frequency plans supported: AS923, EU863-870, US902-928.
  • Battery powered (ER26500), user replaceable.

Manual setup

1
Create an application

In The Things Stack console, go to Applications → Create application and pick a unique application ID.

2
Register an end device

Choose From the LoRaWAN Device Repository and search for “LRS10701 - Indoor Air Quality (IAQ) Sensor”. Select your firmware version and frequency plan.

3
Enter device credentials

Provide the DevEUI, JoinEUI and AppKey printed on the device label or QR code.

4
Power on & verify uplinks

Within a few minutes you'll see the join request and decoded uplinks in the Live data tab.
